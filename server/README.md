# Server

## .env

Add the following rules to your .env file:

* SECRET - The secret used for jsonwebtokens
* ALLOWED_ORIGIN - From which addres can a client connect
* DATABASE_USER
* DATABASE_PASSWORD
* DATABASE_HOST
* DATABASE_PORT

## Setup

### Start server once
```
npm run start
```

### Automatic restarting of the server
```
npm run dev
```
